# SK하이닉스 리포트 수집 → 위키 워크플로

## 권장 수집 채널 (우선순위)
1. **본인 수신함 + Bloomberg/단말 (메인)**
   - 이메일에서 SK하이닉스 리포트 PDF 저장
   - Bloomberg: `000660 KS Equity` → `RES` 함수 → 기간 필터 → PDF 저장
   - 외국계 IB(Nomura/GS/MS/JPM) 원문은 사실상 이 경로가 유일
2. **한경컨센서스** (markets.hankyung.com/consensus) — 국내 중소형 증권사 공백 보완
3. **네이버 금융 리서치 / FnGuide** — 목표주가·투자의견 교차검증

※ 본인 참고용 수동 수집은 정상. 집계 사이트 대량 자동 스크래핑은 약관 위반이니 지양.

## 파일명 규칙 (중요)
PDF를 raw/reports/ 에 넣을 때:
    YYYY-MM-DD_증권사_SK하이닉스.pdf
예: 2026-05-07_미래에셋_SK하이닉스.pdf
→ 스크립트가 날짜·증권사를 자동 인식

## 변환 → Ingest
1. PDF를 raw/reports/ 에 모은다
2. 변환:  python3 pdf_to_md.py
   → raw/reports_md/ 에 .md 생성
3. Claude Code에서:  "raw/reports_md/ 의 ingested:false 리포트를 전부 ingest해줘"
   → 컨센서스 테이블에 행 추가, 상충 견해 갱신, frontmatter ingested:true 로 변경

## 스캔본 PDF (텍스트 추출 안 될 때)
"텍스트 없음" 경고가 뜨면 OCR 필요.
    pip install ocrmypdf  (또는 회사 PDF 도구로 텍스트 레이어 추가 후 재변환)
